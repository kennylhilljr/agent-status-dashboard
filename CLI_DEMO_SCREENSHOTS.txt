              [--refresh-rate REFRESH_RATE]

Unified Agent Status Dashboard CLI with multiple modes

optional arguments:
  -h, --help            show this help message and exit
  --json                Output metrics as JSON to stdout
  --agent NAME          Show detailed view for specific agent
  --leaderboard         Show leaderboard view sorted by XP
  --achievements        Show achievements view (optionally filtered by --agent)
  --once                Run without live updates (applies to dashboard,
                        leaderboard, achievements)
  --metrics-dir METRICS_DIR
                        Directory containing metrics.json (default:
                        ~/.agent_metrics or ./.agent_metrics.json)
  --refresh-rate REFRESH_RATE
                        Refresh rate in milliseconds (default: 500ms, applies
                        to live modes)

Examples:
  cli.py                         # Live dashboard
  cli.py --once                  # Single dashboard update
  cli.py --json                  # Output JSON metrics
  cli.py --agent coding          # Agent detail view
  cli.py --leaderboard           # Live leaderboard
  cli.py --leaderboard --once    # Single leaderboard update
  cli.py --achievements          # Achievements view
